== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `embed_articles.zip` to the `/wp-content/plugins/` directory
2. Unzip `embed_articles.zip`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Visit Settings -> Embed Articles Options to Setup
