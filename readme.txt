== Description ==

<p>The <strong>Embed Articles</strong> wordpress plugin makes <strong>content sharing and embedding</strong> a lot <strong>easier and simpler</strong>!</p>
<p>Shortcode available (<strong>[embed_articles]</strong>)</p>
<p>Example: [embed_articles url=http://www.bbc.co.uk/democracylive/scotland-29461347]</p>
<a href="http://embedarticles.com/how/">How it Works</a>   ~   <a href="http://embedarticles.com/login/">Login</a>   ~   <a href="http://embedarticles.com/contact/">Contact Us</a>
<p>Features:</p>
<ul>
	<li>Auto syndicate your blog to embedarticles.com.</li>
	<li>Share and embed full-text content from around the web to your blog, easily through a shortcode.</li>
	<li>Adds visibility and gets your blog content discovered by other curation and discovery systems through the Open Graph API and Schema.org scopes.</li>
</ul>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `embed_articles.zip` to the `/wp-content/plugins/` directory
2. Unzip `embed_articles.zip`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Visit Settings -> Embed Articles Options to Setup

== Frequently Asked Questions ==

= Do I need an account on embedarticles.com to use this plugin? =

Yes. In order to complete the setup of the plugin you need a API Key from embedarticles.com.  Please use this link to sign up:

<a href="http://embedarticles.com/register/">http://embedarticles.com/register/</a>

It is fast and free.  Once you are registered, visit the Wordpress page under the Integrate section of the site.
