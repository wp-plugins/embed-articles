== Description ==

<p>The <strong>Embed Articles</strong> wordpress plugin makes <strong>content sharing and embedding</strong> a lot <strong>easier and simpler</strong>!</p>
<p>Shortcode available <strong>[embed_articles url=]</strong></p>
<p>Example: [embed_articles url=http://www.bbc.co.uk/democracylive/scotland-29461347]</p>
<a href="http://embedarticles.com/how/">How it Works</a>   ~   <a href="http://embedarticles.com/login/">Login</a>   ~   <a href="http://embedarticles.com/contact/">Contact Us</a>
<p>Features:</p>
<ul>
	<li>Auto generates Open Graph Protocol Metadatas, so you won't bother coding those metadata html codes.</li>
	<li>Auto syndicate your blog to embedarticles.com.</li>
	<li>Share and embed full-text content from around the web to your blog, easily through a shortcode.</li>
	<li>Adds visibility and gets your blog content discovered by other curation and discovery systems through the Open Graph API and Schema.org scopes.</li>
</ul>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `embed-articles.zip` to the `/wp-content/plugins/` directory
2. Unzip `embed-articles.zip`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Visit Settings -> Embed Articles to Setup
